# Enterprise DevOps Platform

A production-ready Employee Management Platform built using:

- Java 21
- Spring Boot
- Spring Cloud
- Spring Security
- Spring Cloud Gateway
- PostgreSQL
- Redis
- Apache Kafka
- Docker
- Kubernetes
- Helm
- Terraform
- GitHub Actions
- AWS

## Architecture

Monolith → Microservices → Eureka Discovery → Kubernetes → AWS
